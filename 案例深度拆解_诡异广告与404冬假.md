## 一、《诡异广告调查事件》（银河电灯）

### 1.1 游戏概况

| 维度 | 说明 |
|------|------|
| 制作方 | 银河电灯（Shiroki） |
| 规模 | 45 层嵌套页面 |
| 玩家角色 | **戴远辰**（江岳市人，"超高校级的幸运"体质） |
| 入口 | 一张 AI 生成的遗照（社交媒体传播） |
| 核心机制 | WebArchive 快照浏览 + 关键词搜索 + 百度街景解谜 + 多层密码 |
| 前作关联 | 《大学生登山失踪事件》同世界观续作（"添福菩萨宇宙"/"基站宇宙"） |
| 游玩提示 | **必须关闭广告屏蔽插件**，否则街景等关键页面无法显示 |

### 1.2 叙事触发器

> 戴远辰在追网剧时，看到一则插播广告——来自一个早在 2008 年就停播的电视台 **"新纪元电视台"**（NewEra TV）。一个已消失十几年的电视台，为什么会出现在今天的广告中？他决定调查。

这个入口设计的精妙之处：
- **日常裂痕**：追剧时看到的广告——最日常的场景中出现无法解释的异常
- **个人视角**：玩家不扮演"侦探"，而是"一个觉得不对劲的普通人"
- **自然动机**：如果你在 2025 年看到 2008 年已停播的电视台广告，你也会想查

### 1.3 全 45 页逐页拆解

我将 45 个页面按**信息获取-解谜-推进**的逻辑链分组拆解。

> - 第一层：九宫格数字 → 拼音（526=看, 3426=电, 9466=影）→ "看电影"
> - 第二层：博文内容中隐藏谜底"日落"→ 拼音"riluo"用作下一篇密码
> - 密码答案来自上一页的内容，而不是外部知识——这是"密码与叙事绑定"的典范

#### 📍 第四幕：隐藏与死亡（P28-P32）⭐ **核心转折**

**目标：发现"三位员工已经死亡"的真相，串联 subliminal（阈下信息）线索**

| 页号 | 页面名称 | 玩家操作 | 获取的信息 | 指向哪里 |
|------|---------|---------|-----------|---------|
| **28** | **赵海涛隐藏页面** | ⭐ 在赵海涛页面搜索框输入密钥 **0mn1516h7** | 黑白页面 + 遗照风格，关键词 **sub** | 暗示组合 subliminal |
| **29** | **曹立刚隐藏页面** | 同上操作输入密钥 | 黑白遗照页，关键词 **limi** | 暗示组合 subliminal |
| **30** | **方睿隐藏页面** | 同上操作输入密钥 | 黑白遗照页，关键词 **nal** | 暗示组合 subliminal |
| **31** | **王振东隐藏页面** | 同上操作输入密钥 | 黑白遗照页，关键词 **ity**（彩蛋向） | 组合 = **subliminal** + ity |
| **32** | **subliminal.html** | ⭐ 在 URL 后加 `/subliminal.html` | **观测者名单**——包含戴远辰、叶凯文等人的名字 | 指向 P33（论坛基站线索） |

> **📌 设计要点**：
> - **密钥来源**：0mn1516h7 是 Leet 语（0=o, m=m, n=n, 1=i, 6=g, 5=s, 7=t）→ **omnisight**（全知视角）
> - **情绪冲击**：四页风格突然从"正常员工页面"骤变为"黑白遗照"——玩家在此刻意识到这些员工已经死亡
> - **拼字解谜**：sub + limi + nal = subliminal（阈下信息/潜意识暗示）——暗示电视台通过阈下广告操控观众
> - **名单页**是游戏第一个"全局揭示"：玩家发现"我（戴远辰）的名字也在名单上"

## 二、《404工位消失事件 2.0：冬假》

### 2.1 游戏概况

| 维度 | 说明 |
|------|------|
| 制作方 | AAA 地下党 |
| 类型 | 模拟 Windows 桌面解谜 / 叙事推理 / 多结局 |
| 平台 | TapTap / 网页端 / PC / 安卓 |
| 玩家角色 | **江奕云**（大二实习生，接手失踪 mentor 的工位电脑） |
| 游戏内时间 | 2026 年 1 月 9 日（周五） |
| 规模 | 约 2-3 小时通关 |

### 2.2 叙事触发器

> 你是一名大二实习生，第一天来到工位，发现你的 mentor **麦闻**（Marvin）失踪了。他的电脑还开着，上面有他留下的所有数字痕迹——微信、邮件、浏览器记录、文件。你需要通过这些碎片拼凑出他失联的真相。

入口设计的精妙之处：
- **"摸鱼友好"**：模拟 Windows 界面，在工位玩也不会引人怀疑
- **零学习成本**：每个人都知道怎么用电脑，不需要"学习如何玩游戏"
- **天然的探索动机**：这是"别人的电脑"——偷看别人隐私的刺激感是天然的驱动力

### 2.3 全系统/页面逐项拆解

与《诡异广告》的"线性页面流"不同，《404 冬假》是一个**模拟操作系统**，多个应用并行可访问。以下按应用维度拆解。

---

#### 💻 系统层：模拟 Windows 桌面

| 元素 | 玩家操作 | 获取的信息 | 指向哪里 |
|------|---------|-----------|---------|

## 三、两案对比总结

### 3.1 核心差异

| 维度 | 诡异广告调查事件 | 404 工位消失事件 2.0：冬假 |
|------|----------------|------------------------|
| **页面形态** | 45 个独立网页，线性浏览 | 模拟操作系统，多应用并行 |
| **导航方式** | 顺序翻页 + 搜索跳转 | 桌面图标 + 自由组合探索 |
| **密码密度** | 高（九宫格/火星文/街景/Leet） | 中（维吉尼亚/日期推算） |
| **现实交叉** | 百度街景真实地点 | 模拟 Windows + 真实校园系统 |
| **叙事节奏** | 悬疑层层递进，终局世界观大反转 | 碎片拼图，终局聚焦现实议题 |
| **结局设计** | 双结局（好坏分明） | 三结局（灰色无完美） |
| **世界观** | 超自然（外星观测者） | 现实向（学术腐败） |
| **彩蛋密度** | 9 个彩蛋 + 前作隐藏结局 | 较少（聚焦主流程） |

### 3.2 可借鉴设计原则

| 原则 | 来源游戏 | 具体含义 |
|------|---------|---------|
| **密码与叙事强绑定** | 两作共同 | 密码答案必须来自游戏世界内部，不能凭空存在 |
| **搜索链自然生长** | 两作共同 | 每个搜索词应该是上一个线索的"自然结果" |
| **界面多样性防疲劳** | 诡异广告 | 45 页不疲惫因为每页视觉风格不同 |
| **跨系统联动** | 404 冬假 | Excel→校内系统→邮箱→门户网站的跨应用链条 |
| **情感灰度结局** | 404 冬假 | 没有完美结局，让玩家通关后继续思考 |
| **虚实结合破次元壁** | 诡异广告 | 百度街景真实地点让"游戏"和"现实"的边界模糊 |
| **"翻别人东西"的驱动力** | 404 冬假 | 偷看电脑/手机是天然的探索动机 |
| **密码类型多样化** | 诡异广告 | 九宫格→拼音→Leet→街景→每个谜题不重复 |

### 3.3 对《夜航》的针对性建议

| 《夜航》可借鉴点 | 来源 |
|----------------|------|
| 搜索链设计（关键词自然生长） | 两作共同 |
| 论坛搜索 + 外部页面跳转的混合模式 | 诡异广告 |
| 密码嵌套（第一层答案用于第二层） | 诡异广告的"看电影→日落"链 |
| 多信息源交叉验证（论坛+外部页面+档案） | 404 冬假的 Excel→邮箱→门户链 |
| 灰色结局增强情感余震 | 404 冬假的三结局 |
| "日常裂痕"入口设计 | 两作共同 |
| 页面视觉多样性（不同年代/风格） | 诡异广告 |
